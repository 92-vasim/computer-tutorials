Photoshop - 3 
    1. Lasso tool
            a. Click & Hold 
            b. Drag 
            c. Join two sides  
    2. Polygonal Lasso tool 
            a. Click & Leave 
            b. Repeat going in direction 
            c. Join two side when the circle is shown 
    3. Magnetic Lasso tool 
            a. Click first 
            b. Keep going ahead 
            c. Join two side when the circle is shown

        d. Add new anchors using Add anchor point tool 
    2. Add anchor point tool 
    3. Paths 
    4. Remove selection = Ctrl + D

    6. Choose make selection option

Photoshop - 6

    I) Types of images:
        1. JPEG: Joint Photographic Expert Group (JPG)
        2. PSD: Photoshop Native Format (Photoshop Document)
        3. BMP: Bitmap 
        4. GIF: Graphic Interchange Format 
        5. PNG: Photographic Network Group 

    II) There are three methods to save a picture in Photoshop:
        1. File --> Save (For PSD)
        2. File --> Save as (For JPG)
        3. File --> Save for web (JPG/PNG)
            File > Export > Save for web 
            1. JPG 
            2. PNG 
            3. WEB  

    III) Clone tool:
        1. Select tool  
        2. Alt + rub = for sample colour 
        3. Right click and increase point/brush size 
        4. Rub in another place
        5. Aligned will be unchecked in option bar
        6. [] = increase & decrease brush size
